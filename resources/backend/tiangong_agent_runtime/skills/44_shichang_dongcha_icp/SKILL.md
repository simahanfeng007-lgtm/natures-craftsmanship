# 市场洞察与ICP定位

面向目标市场、行业细分、ICP、买方画像、痛点和竞争定位场景的技能。目标：明确该卖给谁、为什么买、谁决策、痛点是什么、价值主张如何表达。

## 什么时候用
- 用户要求定位目标客户、分析行业、构建ICP、客户画像、竞品定位时使用
- 用户要求判断某企业是否值得触达、痛点是什么时使用

## 反触发条件（可选）
- 需要公开资料核验时结合联网搜索
- 批量名单评分转交线索识别与评分
- 报价方案转交方案报价与成交推进

## 标准流程
1. 明确业务目标、目标客户、当前阶段、输入数据、风险等级和最终交付物。
2. 先识别客户旅程位置：获客、激活、培育、诊断、方案、成交、复盘。
3. 优先调用只读分析工具形成证据，再生成计划、话术、评分或复盘。
4. 对线索、机会、渠道、内容、触点和成交风险分别输出原因与置信度。
5. 涉及真实触达、批量发送、CRM写入、报价或预算调整时，只生成计划并要求确认。
6. 交付时给出结论、证据、下一步动作、责任人建议、时间节奏和未验证项。
