# 数据表格分析

面向企业名单、Excel、CSV、客户池、财务表、训练记录、线索评分场景的技能。目标：稳定读取、清洗、去重、筛选、打分和导出结构化表格。

## 什么时候用
- 用户上传或提到 Excel、CSV、表格、名单、线索池、客户池时使用
- 用户要求去重、筛选、评分、排序、合并表格、透视汇总时使用
- 涉及企业名单与交付记录比对时使用

## 反触发条件（可选）
- 纯文档长文总结转交文档处理
- 需要真实数据库查询时转交数据库与SQL
- 需要联网找企业信息时转交联网搜索

## 标准流程
1. 明确用户目标、输入材料、风险等级和最终交付物。
2. 判断是否已有足够证据；证据不足时先读取、解析、检索或生成计划。
3. 优先调用只读工具形成证据，再调用写入/执行类工具。
4. 对结果做一致性检查：字段、来源、时间、路径、数量和可复现性。
5. 输出中文结论，区分已验证、推断、风险和未验证项。
6. 如需写入、提交、批量执行或影响外部系统，必须进入确认或只生成计划。
7. 交付时给出结果、证据、产物路径、下一步建议和回滚/复核方式。
