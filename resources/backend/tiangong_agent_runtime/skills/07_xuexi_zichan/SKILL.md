# 学习资产管理

创建/验证/审核/发布学习资产，管理资产契约与沙箱对齐

## 内容生产依赖

学习内容需要公开资料、实时信息、网页材料或事实核验时，先用 `web_search` 获取来源；必要时对关键 URL 使用 `web_readability_extract` 读取正文，再进入学习资产队列和激活流程。
