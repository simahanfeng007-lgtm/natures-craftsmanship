# 综合运营总控

面向从获客到成交的全链路增长运营、销售运营、收入运营统筹场景的技能。目标：把市场、内容、获客、线索、私域、销售、方案、成交、归因和复盘连成一条可执行收入链。

## 什么时候用
- 用户要求做综合运营、增长运营、从获客到成交全链路、销售运营体系时使用
- 用户要求判断当前运营卡点、下一步增长动作、周/月运营计划时使用

## 反触发条件（可选）
- 只写单条销售话术时转交销售作战
- 只分析表格时转交数据表格分析
- 涉及真实触达客户前必须确认

## 标准流程
1. 明确业务目标、目标客户、当前阶段、输入数据、风险等级和最终交付物。
2. 先识别客户旅程位置：获客、激活、培育、诊断、方案、成交、复盘。
3. 优先调用只读分析工具形成证据，再生成计划、话术、评分或复盘。
4. 对线索、机会、渠道、内容、触点和成交风险分别输出原因与置信度。
5. 涉及真实触达、批量发送、CRM写入、报价或预算调整时，只生成计划并要求确认。
6. 交付时给出结论、证据、下一步动作、责任人建议、时间节奏和未验证项。
