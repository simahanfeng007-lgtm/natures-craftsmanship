# 获客渠道与活动运营

面向获客渠道、活动、投放、地推、转介绍、会销、直播、社群和落地页获客场景的技能。目标：选择合适渠道，设计获客活动，估算成本和回收，形成可执行获客计划。

## 什么时候用
- 用户要求做获客、渠道规划、活动策划、投放计划、落地页检查时使用
- 用户要求评估哪个渠道更有效、钱怎么花、线索怎么收集时使用

## 反触发条件（可选）
- 单条内容创作转交内容运营与转化素材
- 线索入池后转交线索识别与评分
- 真实投放/提交表单前必须确认

## 标准流程
1. 明确业务目标、目标客户、当前阶段、输入数据、风险等级和最终交付物。
2. 先识别客户旅程位置：获客、激活、培育、诊断、方案、成交、复盘。
3. 优先调用只读分析工具形成证据，再生成计划、话术、评分或复盘。
4. 对线索、机会、渠道、内容、触点和成交风险分别输出原因与置信度。
5. 涉及真实触达、批量发送、CRM写入、报价或预算调整时，只生成计划并要求确认。
6. 交付时给出结论、证据、下一步动作、责任人建议、时间节奏和未验证项。
