# 天工造物 临渊者 v0.2.0

## 安装（Windows）
下载 **两个分卷文件**到同一目录：
- `天工造物-临渊者-0.2.0-x64-Setup.exe.z01` (75M)
- `天工造物-临渊者-0.2.0-x64-Setup.exe.zip` (49M)

解压（任选其一）：
- **7-Zip**: `7z x 天工造物-临渊者-0.2.0-x64-Setup.exe.zip`
- **WinRAR**: 右键 .zip → 解压到当前文件夹
- **Windows 11 自带**: 资源管理器双击 .zip 自动合并

合并后得到 `天工造物-临渊者-0.2.0-x64-Setup.exe` (125M)，双击安装。

## 源码结构
